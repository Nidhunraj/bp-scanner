# 🔐 Simple Port Scanner

A basic Python port scanner that checks for open ports on a target machine.

## 🚀 Features

* Scans ports 1–1024
* Identifies open ports
* Fast and lightweight

## 🛠️ Requirements

* Python 3

## ▶️ Usage

```bash
python scanner.py
```

Enter the target IP when prompted.

## 📌 Example

```
Enter target IP: 127.0.0.1

Port 80 is open
Port 443 is open
```
